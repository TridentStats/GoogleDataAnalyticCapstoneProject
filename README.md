# Google Data Analytic Capstone Project

This repo contains the R code I used to perform the analysis and create the data visualizations within my Capstone project for the Google Data Analytics Course offered by Coursera.

Most of the code is contained within the finalCapstoneProjectCode.R file, except for all code used to created data visualizations, which is within the CapstoneProjectDataVisualisations.R file. You will have to first run the finalCapstoneProjectCode.R file first in order for the data visualizations file to run successfully.

Follow these steps in order to run both files successfully:
1. Download 'CyclisticTripData' folder, which contains CSV files used for analysis, here: https://drive.google.com/drive/folders/1iR-s0aje4tqz2GScxkx0iEezvKLjl2OI?usp=drive_link
2. If using RStudio, place 'CyclisticTripData' folder in your current working directory.
3. Run 'finalCapstoneProjectCode.R'
4. Run 'CapstoneProjectDataVisuals.R'
